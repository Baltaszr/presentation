Volio
Volio is website builder for all generations to develop website as easy as putting blocks together. Volio simplifying the Website creation platform so people can really focus on what's matter and running your business.

Timeline Set up

Timeline is important, due to let the team is consistent with everything that we are dedicating! So yup even this is an internal project but we straight with the timeline. There are many things that we can dig when we decided to set the timeline. Especially, if the project itself didn’t produce the money at first, it is because a several people that actual passionate about it.

Setting up the timeline, can help us to focus on what matters:

• To make us discipline;
• To make the rules for the team;
• To track the process of making the product

Art Direction

Me and creative director is a fan of simplicity and minimalism, so after we decide to set the timeline, then we find some mood-boards to create the art direction for Volio. Especially the look and feel for the web app, also the whole brand identity of Volio.

Design System

After we decided the art direction and know what we wanted to achieve and set the goals for this project. Then create the design system to implement what kind of looks that we are trying to create also how we came with new design possibilities that we crafted from scratch.

On this design system we implemented atomic design methodology. The purpose of this method is to breakdown every components into atoms, molecules, organism, templates, and pages, so that it would provide the team clear explanation on every components. And the most important is to create effective when we start build the product and update the design.

Result

The product is driven by our curiosity to provide every professional and entrepreneur for creating their website as easy as they wish, without knowing programming skill or design skill.
